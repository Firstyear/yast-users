YaST2: Users configuration module

Features (SL 8.2)

Basic module abilities:

  • add, remove and edit local users
  • add, remove and edit local groups
  • read NIS users and groups
  • add, remove and (partial) edit of LDAP users and groups
  • configure password encryption type
  • configure authentication type
  • edit defaults from /etc/defaults/useradd
  • password changing and warning limits
User attributes:
  • name
  • password
  • uid (UID currently cannot be changed)
  • GECOS
  • group (groups)
  • shell
  • home directory
  • password settings (warning limits etc.)
Group attributes:
  • name
  • gid
  • users
  • password access

Changes from 8.1

  • major speedup, see benchmark
  • LDAP support
  • editing defaults ( bug #14129)
  • expert dialog (authentication and password encryption)
  • editing UID ( bug #16084)

Current workflow (can be found at http://lide.suse.cz/~~jsuchome/users/workflow.png):

1.1 Dialogs

  • Summary dialog (current screenshot):

    This dialog provides multiple views to sets of users (local/system/...). So there is a Menubutton for seting such view. New "Customize view" button will open the Customize popup.

  • Defaults dialog ( current screenshot)

    Here the deafault settings for new users, such as default group, default shell, etc. are editable (bug #14129).

  • Customize popup ( current screenshot)

    In this popup user can customize the view of the users set. (He can e.g. set to view only local and NIS users.)
